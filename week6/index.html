Already being familiar with forms in HTML, here are ways to implement them into JS when calling the class.

document.form[name/index] allows us to create the form.

    input.value or textarea.value allows for there to be a new value or text.

    input.checked allows there to be a checkbox or radio button

    select allows for there to be 3 properties: .option, .value, and .selectedIndex. Including .checked adds a checkbox or radio button.

Form Data

Form data object replicates what a form does and sends data through a fetch call.

Inside the array there is one array for each element. Those include a name and a value.

With the .append method you can put the name and value inside to send data to the form.

When you want to upload a file it is a .blob

.fileName is what you want to call the file that is being sent to the server.

Forms Submit

The submit event happens when the form is submitted and validates the form before sending it to the server.

The event allows for the user to be alerted is input isn't put correctly or if nothing is submitted at all.

Enter can used as long as the submit button is focused even though there is no 'click' involved.

The form.submit() method is called to submit a form to the server manually.
