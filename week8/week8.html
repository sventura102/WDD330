JSON Web Tokens

While reading the article, I found that JWTs are important and are used a lot more than I think. It's when the client [the user] wants to access something on the website but the server, for example, wants them to log in in order to verify that they have access to a specific part of the website. Kind of like a free trial versus a paying user experience. So the user puts in their username and password, then that goes to the server, the server checks if the information received it valid and from there will decide if they grant the client access. If not, an error will send. If they're granted access, they get permissions to access a different experience.
